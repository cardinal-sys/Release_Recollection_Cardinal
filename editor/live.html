◆ GITHUB SYNC ── 記憶の刻印

Live Sync Conduit で読み取ったキーマップを GitHub リポジトリへ直接コミットします。
ブラウザから GitHub API を呼び出すため、ローカルサーバ等の追加ツールは不要です。

Settings → Developer settings → Personal access tokens → Fine-grained tokens
必要な権限: contents: read/write（対象リポジトリのみ）
