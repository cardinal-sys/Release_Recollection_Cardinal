◆ Gestures Editor — 剣技の編纂

各方向の bindings は 2 つ： Default（修飾キーなし）と Mod-pressed（修飾キー押下時）。 例: &kp LG(C)（コピー）/ &kp LG(X)（カット）

◆ Macros Editor — マクロ術式

各マクロは bindings sequence。&macro_press / &macro_tap / &macro_release で送出制御。 one-param / two-param 系では MACRO_PLACEHOLDER をそのまま残す。

◆ Behaviors Editor — 武装完全支配術

各 behavior の compatible で種別が決まる： hold-tap / sticky-key / sensor-rotate / mod-morph / tri-state 等。 properties（flavor / tapping-term-ms 等）は behavior 種別ごとに異なる。

◆ Combos Editor — 同時押し術式

各コンボは name + bindings + key-positions の三要素。 key-positions は物理キー位置のインデックス（数値、空白区切り）。bindings は ZMK behavior 表記 （例: &kp ESCAPE, &mkp MB1, &tog 5）。
